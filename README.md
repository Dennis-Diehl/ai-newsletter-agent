# AI Newsletter Agent

An autonomous agent that researches the latest news from leading AI companies, summarizes the most relevant articles, and generates a formatted HTML newsletter, which gets delivered straight to your inbox.

Built with **LangGraph** for pipeline orchestration, **Tavily** for AI-powered search, **Playwright** for web scraping, **Google Gemini 2.5 Flash** for scoring, summarization, and newsletter writing, and **Streamlit** for the UI.
