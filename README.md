# AI Newsletter Agent

An autonomous agent that researches the latest news from leading AI companies, summarizes the most relevant articles, and generates a formatted HTML newsletter, which gets delivered straight to your inbox.

Built with **LangGraph** for pipeline orchestration, **Tavily** for AI-powered search, **Playwright** for web scraping, **Groq (Llama 3.3 70B)** for summarization, **Google Gemini 2.0 Flash** for writing, and **Streamlit** for the UI.
